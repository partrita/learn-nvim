# 복사, 붙여넣기 및 비주얼 모드
Vim의 복사 방식은 일반적인 편집기와 조금 다릅니다. OS 클립보드로 바로 복사하는 대신, `레지스터(Register)`라는 내부 저장 공간을 거칩니다. 물론 원한다면 시스템 클립보드와 연동할 수도 있습니다.

## 레지스터를 이용한 복사 및 붙여넣기
텍스트를 복사(**Yank**)하려면 일반 모드에서 `y` 연산자와 이동 명령어를 조합하거나, 비주얼 모드(`v`)에서 영역을 선택한 뒤 `y`를 누르면 됩니다.

**주의**: Vim에서는 텍스트를 `변경(c)`하거나 `삭제(d)`할 때도 해당 내용이 자동으로 레지스터에 저장됩니다. 이는 삭제한 내용을 나중에 다시 붙여넣을 수 있게 하려는 의도입니다.

복사한 내용을 붙여넣는 방법은 다음과 같습니다.
* `p`: 커서 뒤에 붙여넣기
* `P`: 커서 앞에 붙여넣기
* `Alt + p` (또는 `<M-p>`): 입력 모드에서 즉시 붙여넣기

## 비주얼 모드 (Visual Mode)
마우스로 드래그하는 것처럼 텍스트 영역을 선택할 때 사용합니다.
* `v`: 일반 비주얼 모드 (글자 단위)
* `V`: 비주얼 라인 모드 (줄 단위)
* `Ctrl + v`: 비주얼 블록 모드 (세로 블록 단위)

비주얼 모드에서 영역을 선택한 뒤 새 내용을 붙여넣으면, 선택된 영역이 교체됩니다. 이때 기존에 있던 내용은 자동으로 레지스터에 복사됩니다.

## 레지스터 (Registers) 활용하기
특정 레지스터를 지정하려면 `"` 키를 사용합니다. 
예를 들어 `a`라는 이름의 레지스터에 복사하고 싶다면 `"ay<이동>`을 입력하면 됩니다.

붙여넣을 때도 마찬가지로 `"ap`와 같이 입력해 특정 레지스터의 내용을 가져올 수 있습니다.

텍스트를 어디에도 남기지 않고 완전히 삭제하고 싶을 때는 `_`(블랙홀 레지스터)를 활용합니다. (예: `"_dd`)

레지스터는 활용도가 무궁무진합니다. 더 자세한 내용은 `:help registers`를 통해 확인해 보세요.

### 유용한 단축키 설정 예시
삭제하거나 변경할 때 기존 내용을 보존하고 싶다면 다음과 같은 커스텀 단축키를 추천합니다.
```lua
-- 복사 없이 교체
vim.keymap.set('x', '<leader>p', '"_dP', { desc = '기존 내용 보존하며 붙여넣기' })

-- 복사 없이 삭제 및 변경
vim.keymap.set('n', '<leader>d', '"_d', { desc = '복사 없이 삭제' })
vim.keymap.set('n', '<leader>c', '"_c', { desc = '복사 없이 변경' })
```

## 시스템 클립보드 연동
시스템 클립보드와 통신하는 특수 레지스터는 `+`입니다. 이를 활용하면 외부 프로그램과 텍스트를 주고받을 수 있습니다.

매번 `"+`를 입력하는 건 번거로우므로 다음과 같이 단축키를 설정해 두면 편리합니다.
```lua
-- 리더 키를 활용한 클립보드 연동
vim.keymap.set({'n', 'v'}, '<leader>y', '"+y', { desc = '시스템 클립보드로 복사' })
vim.keymap.set('n', '<leader>p', '"+p', { desc = '시스템 클립보드에서 붙여넣기' })
```

## 추천 플러그인
* [yanky.nvim](https://github.com/gbprod/yanky.nvim): 붙여넣기 후 이전 복사 기록을 빠르게 순환하며 선택할 수 있게 해줍니다.
* [peekup](https://github.com/gennaro-tedesco/nvim-peekup): 레지스터의 내용을 시각적으로 확인하고 선택할 수 있습니다.
* [registers.nvim](https://github.com/tversteeg/registers.nvim): 단축키 입력 시 레지스터 목록을 팝업으로 보여줍니다.
