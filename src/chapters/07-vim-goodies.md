# Vim의 유용한 기능
Vim에는 단순한 텍스트 입력을 넘어 편집 효율을 극대화해 주는 강력한 기능들이 숨어 있습니다. 그중에서도 실무에서 자주 쓰이는 핵심 기능들을 알아보겠습니다.

---

## 퀵픽스 목록 (Quickfix List)
퀵픽스 목록은 파일명, 위치(줄, 칸), 그리고 해당 위치의 텍스트 정보를 담은 일종의 '작업 목록'입니다. 주로 코드 검색 결과나 컴파일 오류 목록을 한눈에 보고 빠르게 이동할 때 사용합니다.

### 주요 명령어
* `:copen`: 퀵픽스 창을 엽니다.
* `:cclose`: 퀵픽스 창을 닫습니다.
* `:cnext`: 목록의 다음 항목으로 이동합니다.
* `:cprev`: 목록의 이전 항목으로 이동합니다.

항목 간 이동을 더 편하게 하려면 [vim-unimpaired](https://github.com/tpope/vim-unimpaired) 플러그인을 설치해 `]q`(다음), `[q`(이전) 단축키를 쓰는 걸 추천합니다.

---

## 치환 명령 (Substitute)
현재 편집 중인 파일에서 특정 문자열을 찾아 다른 내용으로 바꿀 때 사용합니다. 기본 구문은 리눅스의 `sed` 명령어와 매우 비슷합니다.

### 활용 예시
* `:s/old/new/`: 현재 줄에서 가장 처음 발견된 `old`를 `new`로 바꿉니다.
* `:s/old/new/g`: 현재 줄의 모든 `old`를 `new`로 바꿉니다.
* `:%s/old/new/g`: 파일 전체에서 모든 `old`를 `new`로 바꿉니다.

더 자세한 옵션은 `:help :substitute`에서 확인할 수 있습니다.

### 편리한 이름 바꾸기(Rename) 단축키
커서 아래의 단어를 즉시 치환하고 싶을 때 유용한 설정입니다.
```lua
-- 리더 + F2로 현재 단어 전체 치환 준비
vim.keymap.set('n', '<leader><F2>', '*:%s///g<left><left>', { desc = '단어 일괄 치환' })
```

## 전역 명령 (Global Command)
특정 패턴이 포함된 줄을 필터링하여, 해당 줄들에만 일괄적으로 명령을 내릴 때 사용합니다.

* `:g/pattern/d`: `pattern`이 포함된 모든 줄을 한꺼번에 삭제합니다.

복잡한 텍스트 정리 작업에서 상상 이상의 위력을 발휘합니다. 자세한 내용은 `:help :global`을 참고해 보세요.

---

## 매크로 (Macros)
반복되는 일련의 작업을 녹화해서 그대로 재생하는 기능입니다. 노가다성 작업을 순식간에 끝내주는 Vim의 백미라고 할 수 있습니다.

1. **녹화**: `q`를 누른 뒤 저장할 알파벳(예: `a`)을 누릅니다. 왼쪽 하단에 `recording @a`가 표시됩니다.
2. **작업**: 반복할 편집 과정을 한 번 수행합니다.
3. **종료**: 다시 `q`를 눌러 녹화를 마칩니다.
4. **재생**: `@a`를 눌러 실행합니다. 숫자와 조합해 `10@a`처럼 입력하면 10번 연속으로 실행됩니다.

### 활용 사례: 요일 목록 변환하기
다음과 같은 C 스타일 enum을 파이썬 리스트 형식으로 바꿔야 한다고 가정해 봅시다.

```c
Sunday = 0,
Monday = 1,
...
```

매크로(`qa^guui'<Esc>ea'<Esc>ldt,jq`)를 한 번 녹화해두면, 아무리 긴 목록이라도 눈 깜짝할 새에 아래와 같이 바꿀 수 있습니다.

```python
'sunday',
'monday',
...
```

## 점(.) 명령으로 마지막 작업 반복
방금 수행한 편집 작업을 바로 다음 위치에서 반복하고 싶을 때 `.` 키를 누르세요. 아주 간단한 매크로라고 생각하면 이해하기 쉽습니다.

예를 들어, 어떤 단어를 `ciw`로 수정한 뒤 다른 단어로 이동해 `.`만 누르면 똑같은 수정이 적용됩니다.

---

## 외부 명령 실행 (External Command)
편집기를 나가지 않고도 터미널 명령을 실행할 수 있습니다.
* `:!ls`: 현재 디렉토리의 파일 목록을 보여줍니다.

---

## 코드 접기 (Folding)
긴 코드를 구조별로 접어서 가독성을 높여주는 기능입니다. 
자동으로 코드가 접히는 게 번거롭다면 다음 설정을 통해 비활성화할 수 있습니다.
```lua
vim.opt.foldlevelstart = 99 -- 시작 시 모든 코드 펼치기
```
관심이 있다면 [nvim-ufo](https://github.com/kevinhwang91/nvim-ufo) 같은 플러그인으로 훨씬 미려한 접기 기능을 구현해 보세요.
