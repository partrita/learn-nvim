# UI 구성하기
Neovim의 가장 매력적인 점 중 하나는 편집기가 고정된 네이티브 UI를 강요하지 않는다는 것입니다. 대신 UI를 마음대로 주무를 수 있는 강력한 프레임워크를 제공하죠. 덕분에 커뮤니티에서는 상상도 못 할 만큼 멋지고 다양한 UI 플러그인들이 쏟아져 나오고 있습니다.

---

## 핵심 구성 요소

### 색상 테마 (Colorscheme)
가장 먼저 정해야 할 건 역시 눈이 편안한 테마겠죠. 정말 수많은 선택지가 있습니다. 저는 직접 만든 [ofirkai.nvim](https://github.com/ofirgall/ofirkai.nvim)을 사용하지만, 여러분은 이미 익숙한 테마를 먼저 찾아보시는 걸 추천합니다.

[유명한 색상 테마 목록](https://github.com/rockerBOO/awesome-neovim#colorscheme)에서 마음에 드는 걸 골라보세요.

### 상태 표시줄 (Statusline) & 창 표시줄 (Winbar)
상태 표시줄은 명령줄 바로 위, 하단에 위치한 줄입니다. 현재 모드, 파일명, 파일 형식 등 원하는 정보를 자유롭게 표시할 수 있습니다.

상태 표시줄은 크게 두 가지 방식으로 설정할 수 있습니다.
* **전역(Global)**: 현재 활성화된 버퍼의 정보만 보여주는 하나의 공통 줄을 사용합니다.
* **로컬(Local)**: 분할된 각 창마다 별도의 줄을 가집니다. 활성/비활성 창에 따라 디자인을 다르게 할 수도 있죠.

최근에는 창 상단에 위치하는 **창 표시줄(Winbar)**도 많이 쓰입니다. 각 창의 경계가 더 명확해지고 유용한 정보를 상단에 띄울 수 있어 편리합니다.

추천 플러그인으로는 [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim)이 가장 대중적이며, 더 가벼운 [feline.nvim](https://github.com/feline-nvim/feline.nvim)도 인기가 많습니다.

전역/로컬 설정 방법:
```lua
vim.opt.laststatus = 3 -- 전역 설정 (추천)
vim.opt.laststatus = 2 -- 각 창마다 개별 표시 (기본값)
```

### 탭 표시줄 (Tabline / Bufferline)
웹 브라우저처럼 상단에 열려 있는 파일 목록을 보여주는 줄입니다.

주요 플러그인:
* [bufferline.nvim](https://github.com/akinsho/bufferline.nvim): 가장 기능이 풍부하고 미려합니다. 저도 애용 중입니다.
* [barbar.nvim](https://github.com/romgrk/barbar.nvim): 깔끔하고 세련된 디자인으로 인기가 많습니다.

### 파일 탐색기 (File Explorer)
Neovim에도 내장 탐색기(`netrw`)가 있지만, 대부분의 사용자는 더 직관적인 플러그인을 선호합니다.

추천 파일 트리:
* [nvim-tree.lua](https://github.com/nvim-tree/nvim-tree.lua): 설정이 쉽고 필요한 기능이 다 들어있습니다.
* [neo-tree.nvim](https://github.com/nvim-neo-tree/neo-tree.nvim): 최신 기능을 무장한 차세대 탐색기입니다.
* [oil.nvim](https://github.com/stevearc/oil.nvim): 파일을 텍스트 버퍼처럼 편집할 수 있는 혁신적인 탐색기입니다.

UI 설정은 개인의 취향이 가장 많이 반영되는 부분입니다. 여러 조합을 시도해 보면서 여러분에게 가장 영감을 주는 작업 환경을 만들어 보세요.
